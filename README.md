# ラクガキクエスト v0.4.0 UAT

自分で描いた主人公と、手描きの敵が戦うスマホ縦型の育成・放置オートバトルです。外部APIやサーバーは不要で、GitHub Pagesへそのまま公開できます。

## v0.4.0 の更新

- 絵文字だった敵を、Canvasで描くオリジナルの落書きイラストへ全面変更
- ホチキスガニ、ふせんコウモリ、定規ナイトなど、名前由来の文房具パーツを追加
- 敵図鑑40体を追加。未遭遇はシルエット表示
- 通常、赤インク変異、でかラクガキ、走り書き、インク暴走の個体差を追加
- 遭遇回数と発見した個体バリエーションを図鑑へ記録
- ひらめきを9種から15種へ拡張
- すぐ試せるサンプル主人公6体を追加（選択後の描き足しにも対応）
- 「スケッチ広場」を「経験値スケッチ広場」へ改称し、配置方法と効果を明記
- 所持資源が足りない強化ボタンを薄くし、押せない状態を明示
- ブラウザのURLが出ないゲーム内確認ダイアログへ変更
- UAT用に放置報酬を1000倍へ設定

## 既存機能

- 10章・全100ステージ、10体のボス
- 現在戦力、ステージ推奨戦力、危険表示
- クリア済みステージの自由選択・再挑戦
- 6種類の武器、装備切替、レベル強化
- 主人公レベルと攻撃・HP・会心の永久育成
- 最大12時間分を蓄積する放置施設
- 拠点 Lv.1 / MAX 20 の明示
- 初回チュートリアル、お絵描き中のスクロール防止、画面拡大防止
- 端末内自動保存、PWA、旧バージョンのセーブ引き継ぎ

## GitHub Pagesで公開

1. このフォルダ内のファイルをGitHubリポジトリ直下へアップロードします。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Deploy from a branch`、`main`、`/ (root)` を選びます。
4. 数分後に表示される公開URLを開きます。

`index.html` を直接開いても基本機能は動きます。ホーム画面追加やオフラインキャッシュの確認には、GitHub PagesなどのHTTPS環境を使用してください。

## UAT倍率を本番へ戻す

`app.js` 冒頭の `UAT_REWARD_MULTIPLIER` を `1000` から `1` に変更してください。
