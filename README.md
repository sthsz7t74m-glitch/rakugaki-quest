# ラクガキクエスト v0.6.1 UAT

自分で描いた主人公と手描きの敵が戦う、スマホ縦型の育成・放置オートバトルです。外部APIやサーバーは不要で、GitHub Pagesへそのまま公開できます。

## UAT設定

放置報酬は確認しやすいよう1000倍です。本番へ戻す場合は `app.js` 冒頭の `UAT_REWARD_MULTIPLIER` を `1000` から `1` に変更してください。

## GitHub Pagesで公開

1. このフォルダ内のファイルをGitHubリポジトリ直下へ上書きアップロードします。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Deploy from a branch`、`main`、`/ (root)` を選びます。
4. 数分後に表示される公開URLを開きます。

`index.html` を直接開いても基本機能は動きます。ホーム画面追加やオフラインキャッシュにはGitHub PagesなどのHTTPS環境を使用してください。
