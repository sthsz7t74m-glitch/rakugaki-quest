# ラクガキクエスト v0.3.0

自分で描いた主人公が戦う、スマホ縦型の育成・放置オートバトルです。外部APIやサーバーは不要で、GitHub Pagesへそのまま公開できます。

## 主な機能

- 10章・全100ステージ、10体のボス
- 棒人間、ゴブリン、ドラゴンなど26系統の敵デザイン
- 敵ごとにHP・攻撃・速度・回避・装甲など異なる特性
- 現在戦力、ステージ推奨戦力、危険表示
- クリア済みステージの自由選択・再挑戦
- 6種類の武器、装備切替、レベル強化、ボス撃破による解放
- 主人公レベルと攻撃・HP・会心の永久育成
- 最大12時間分を蓄積する放置施設（インク・素材・経験値・宝探し）
- 拠点 Lv.1 / MAX 20 と配置枠の明示
- 初回チュートリアルと「？」からの再確認
- お絵描き中のスクロール防止、連打・ピンチによる拡大防止
- 端末内自動保存、PWA、旧バージョンのセーブ引き継ぎ

## GitHub Pagesで公開

1. このフォルダ内のファイルをGitHubリポジトリ直下へアップロードします。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Deploy from a branch`、`main`、`/ (root)` を選びます。
4. 数分後に表示される公開URLを開きます。

`index.html` を直接開いても基本機能は動きます。ホーム画面追加やオフラインキャッシュの確認には、GitHub PagesなどのHTTPS環境を使用してください。
